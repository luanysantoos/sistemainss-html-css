Manutenção Básica do Computador

Limpeza de Arquivos

  • Arquivos Temporários: Windows + R - digite %temp% - Delete tudo
  • Limpeza de Disco: Pesquise "Limpeza de Disco" - Selecionar C: - Limpar
  • Prefetch: Windows + R - digite Prefetch - Delete tudo
  • Recent: Windows + R - digite recent - Delete tudo

Atualizações

  • Windows Update: Configurações - Atualização e Segurança
  • Verificar Atualizações: Clique em "Verificar se há atualizações"
  • Reiniciar após atualizar: Sempre reinicie o computador

Desfragmentação (HDD)

  • Pesquise: "Desfragmentar e Otimizar"
  • Selecionar: Escolha o disco C:
  • Otimizar: Clique em "Otimizar"

Comandos e Atalhos do Windows

Atalhos Essenciais

  • Windows + E - Abrir Explorador de Arquivos
  • Windows + R - Executar comando
  • Windows + I - Configurações
  • Windows + L - Bloquear computador
  • Windows + D - Mostrar área de trabalho
  • Windows + X - Menu de acesso rápido

Executar (Windows + R)

  • control - Painel de Controle
  • msconfig - configuração do Sistema
  • devmgmt.msc - Gerenciador de Dispositivos
  • services.msc - serviços do Windows
  • appwiz.cpl - Programas e Recursos
  • ncpa.cpl - Conexões de Rede
  • sysdm.cpl - Propriedade do Sistema

Modo Seguro

  • Método 1: Shift + Reiniciar - Solução de Problemas - Inicialização Avançada
  • Método 2: msconfig - Inicialização Segura
  • Método 3: Forçar 3 desligamento no boot

Comandos do Prompt (CMD)

Como abrir: Windows + R → digite cmd - Enter (ou clique direito "Executar como administrador")

Sistema

  • Systeminfo - info completa
  • hostname - nome do PC
  • whoami - usuário atual
  • tasklist - Lista de processos
  • taskkill /im programa.exe /f - fechar forçado
  • shurtdown /r /t 0 - reiniciar
  • shurtdown /s /t 0 - desligar
  • sfc / scannow - verificar arquivos
  • chkdsk C: /f /r - verificar disco

Rede

  • ipconfig
  • ipconfig /all - Ver todas as configuração
  • ipconfig /release - Liberar IP
  • ipconfig /renew - Ronovar IP
  • ipconfig /flushdns - Limpar cache DNS
  • ping 8.8.8.8 - Testar conexão internet
  • ping servidor - Testar Conexão com servidor
  • nslookup site.com - Consultar DNS
  • tracert site.com - Traçar rota
  • netstat -an - Conexões ativas

Arquivos e Pastas

  • dir - listar arquivos
  • cd pasta - Entrar em pasta
  • cd .. - Voltar pasta
  • mkdir nome - Criar pasta
  • del arqivo - Deletar arquivo
  • copy origem destino - Copiar arquivo
  • move origem destino - Mover arquivo
  • ren antigo novo - Renomear

Configurações de Rede

Diagnóstico Rápido

  • Sem internet:
  • 1. Verificar Cabo/WI-FI
  • 2. Verificar se WI-FI está ligado
  • 3. Reiniciar reteador
  • 4. Executar: ipconfig /release e ipconfig /renew
  • Lentidão: ping 8.8.8.8 -t para monitorar
  • DNS: ipconfig /flushdns

Configurar Rede

  • Abrir conexões: Windows + R - ncpa.cpl
  • Ver IP: Clique direito na conexão → Status → Detalhes
  • Configurar IP: Propriedades → IPv4 → Propriedades
  • DNS Google: 8.8.8.8 e 8.8.4.4

Mapeamento de Unidades

  • Explorador - Computador - Mapear Unidade de Rede
  • Caminho: \servidor\pasta
  • Reconectar: Marcar "Reconectar ao fazer logon"
  • CMD: net use Z: \servidor\pasta

Microsoft Office 365

Outlook

  • Criar regra: Página Inicial - Regras - Criar Regra
  • Ausência: Arquivo - Reposta Automáticas
  • Assinatura: Arquivo - Opções - Email - Assinaturas

Segurança da Informação

Boas Práticas

  • nunca compartilhe senha
  • Bloqueie o PC: Windows + L
  • Não abra anexos suspeitos
  • Verifique rementente antes de clicar
  • Senha fortes sempre
  • Troque senhas periodicamente

Antivírus

  • Verificação rápida: Segurança do Windows - Proteção contra vírus
  • Verificação completa: Opções de verificação - Verificação completa
  • Atualizar: Atualizações de proteção

Backup

  • Salvar docs importantes na rede
  • Usar OneDrive
  • Evitar armazenar tudo no PC

Problemas Comuns e Soluções

Computador Lento

  • Abrir Gerenciador de Tarefas: Ctrl + Shift + Esc
  • Verificar processos com alto uso de CPU/Memória
  • Limpar arquivos temporários
  • Desativar programas na inicialização
  • Reiniciar o computador

Impressora não Funciona

  • Verificar se está ligada e conectada
  • Verificar fila de impressão: control printers
  • Cancelar documentos parados
  • Reiniciar serviço de spooler:
  • 1. services.msc
  • 2. Localizar "Spooler de Impressão"
  • 3. Clique direito - Reiniciar

Programa não Abre

  • Tentar executar como administrador
  • Verificar se o programa está atualizado
  • Reiniciar o computador
  • Reparar pelo Painel de Controle
  • Reinstalar o programa

Tela Azul (BSOD)

  • Anotar o código de erro
  • Reiniciar o computador
  • Executar sfc /scannow como administrador
  • Verificar atualizações do Windows
  • Atualizar drivers
  • Contatar o suporte de TI

Sem Áudio

  • Verificar se não está no mudo
  • Clique direito no ícone de som → Abrir configurações
  • Verificar dispositivo de saída correto
  • Atualizar driver de áudio

Esqueci a Senha

  • Contatar o suporte de TI
  • Solicitar reset de senha
  • Aguardar validação

Contatos do Suporte de TI

  • Informar localização da estação

Informações Importantes

  • Patrimônio: Etiqueta no gabinete
  • Nome do PC: hostname no CMD
  • IP: ipconfig no CMD
  • Usuário: whoami no CMD
